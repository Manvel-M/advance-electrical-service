# Hello World

Welcome to my new blog built with Astro. Because Astro renders to static HTML, this page loads incredibly fast.

## Why Content Collections Rule

- **Type Safety:** Built-in Zod validation means no broken frontmatter.
- **Performance:** Content is cached and optimized during the build process.
- **Flexibility:** You can easily swap Markdown for MDX later if you want to embed interactive components.
